# STM32-Study_table
该项目是为别人做的毕业设计，是智能学习桌
## 功能简介
刷卡后语音播报开始学习，同时显示桌面水平倾角，并且检测是否发生剧烈震动，若发生则语音播报语音警告
## 所需硬件
- MPU6050
- RFID模块MFRC522
- STM32F103C8T6单片机
- OLED显示屏
- 语音播报模块dysv5w
- 震动传感器SW420
## 引脚连接
### OLED屏部分
- D1---PA7
- D0----PA5
- RES---PB0
- DC-----PB1
- CS-----PA4
- VCC---3.3V
- GND----GND
### RFID模块
- VCC---3.3V
- RST----PB10
- GND----GND
- IRQ----悬空
- MISO---PB14
- MOSI---PB15
- SCK----PB13
- SDA----PN12
### 震动传感器
- VCC---3.3V
- D0-----PA3
- GND----GND
### MPU6050
- VCC---3.3V
- SCL----PB6
- SDA----PB7
- GND---GND
- 其余悬空
